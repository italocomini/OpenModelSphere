Introduction

Thank you for using Open ModelSphere. The ModelSphere applications are created in a multiplatform development environment. Therefore, screen images in the manuals can be slightly different from what your graphic interface will show. There are actually aesthetic differences (windows, sub-menus, buttons, etc.) between Windows, Linux and Solaris, but the ModelSphere operation is totally identical from one platform to another.
