Installation et exigences minimales
 

Open ModelSphere peut être installé sur les systèmes d'exploitation suivants :

  • Microsoft Windows 98, Millenium, NT4, 2000, XP et Vista
  • Linux
  • Solaris

Installation

Lancez le fichier exécutable de l’application et suivez les instructions contenues dans le fichier Lisezmoi.txt. lancez l’application pour entrer le numéro de licence.

La Machine Virtuelle (VM) Java

L’application est conçue pour fonctionner sur la machine virtuelle de Java 2 (version 5.0). L’application est installée avec la VM 5.0 de Sun Microsystems. La machine virtuelle ne se heurte pas aux autres applications déjà installées sur le poste de travail parce qu’elle est installée séparément dans le répertoire de l’application.

Répertoires d'application
 
RÉPERTOIRE DESCRIPTION
<<user directory>> Le répertoire de l'utilisateur dans le système d’exploitation. Contient les fichiers de configuration qui ont été installés ou générés.
<<application directory>> Le répertoire où l'application est installée. Contient les fichiers d'application qui ont été installés ou générés.
