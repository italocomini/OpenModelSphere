Installation and Requirements


It is possible to run Open ModelSphere on any one of the following operating systems:
  • Microsoft Windows 98, Millenium, NT4, 2000, XP and Vista
  • Linux
  • Solaris

Installation

Run the installation executable file and follow the instructions in the Readme.txt. Start the application to enter the license code.

Java Virtual machine (VM)

The application is designed to run in a Java 2 compliant VM (Version 5.0). The application is installed with Sun’s Java Runtime Environment 5.0. The virtual machine does not interfere with others already installed on the workstation because it is installed in private with the application.

Application directories

DIRECTORY DESCRIPTION
<<user directory>> The operating system’s user home directory. Contains installed/generated application’s configuration files.
<<application directory>> The directory where the application is installed. Contains installed/generated application files.
