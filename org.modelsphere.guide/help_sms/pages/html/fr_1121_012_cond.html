La classe COND


Paramètre : expression booléenne.

Modificateurs facultatifs : TRUE, FALSE, PREF, SUF, NULL.

Algorithme :
Si la condition est vraie, il exécute la règle spécifiée par TRUE. Dans le cas contraire, il exécute la règle indiquée par FALSE.
Si la chaîne résultante est vide, il exécute la règle indiquée par NULL et retourne le résultat de NULL. Dans le cas contraire, il exécute la règle indiquée par PREF et préfixe la chaîne résultante avec le résultat de PREF; ensuite il exécute la règle indiquée par SUF et suffixe la chaîne résultante avec le résultat de SUF.

Exemple
create COND BOOLEAN (uppercase)
TRUE=CREATE
FALSE=create;

où uppercase est une variable booléenne.
