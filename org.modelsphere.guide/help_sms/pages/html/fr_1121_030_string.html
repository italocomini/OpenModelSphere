Expressions de chaînes de caractères


Une expression de chaînes de caractères est une expression qui résulte en une chaîne. Ceci inclut des littéraux de chaînes et des variables de chaînes. Voici des expressions de chaînes valides :

"A character string" //littéraux de chaînes
STRING(stringVar1) //utilisation d'une variables de chaînes (préalablement définie)

Seules les expressions de chaînes de caractères peuvent être assignées aux variables de type chaîne :

setStringVar2 SET(stringVar2, "A character string");
setStringVar2 SET(stringVar2, STRING(stringVar1));
