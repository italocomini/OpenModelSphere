Comment démarrer l'application
==============================

1) Assurez-vous d'avoir installé la machine virtuelle java (version 1.6.30 ou supérieure).


2) Éditez le fichier 'modelsphere.bat' pour vous référer à votre machine virtuelle :
   e.g. "C:\Program Files\Java\jre1.6.0_31/bin/javaw.exe"

3) Pour démarrer Open ModelSphere, double-cliquez sur le fichier 'modelsphere.bat'.


Vous pouvez aussi utiliser la ligne de commande du fichier 'modelsphere.bat' pour créer 
un raccourci pour le démarrage de l'application. L'icone 'OpenModelSphere.ico' est 
disponible pour y être associé au raccourci.
