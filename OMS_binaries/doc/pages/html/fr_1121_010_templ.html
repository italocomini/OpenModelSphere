La classe TEMPL


Paramètre : chaîne de caractères représentant un gabarit.

Modificateurs facultatifs : PREF, SUF, NULL, DOM, EXTERN, FILE, UPPER, LOWER.

Algorithme :
TEMPL exécute la chaîne de gabarit.
Si DOM est spécifié, il traduit la chaîne résultante comme étant spécifiée dans le domaine. Si la chaîne résultante est vide, il exécute la chaîne de gabarit indiquée par NULL et rend le résultat de NULL. Dans le cas contraire, il exécute le gabarit indiqué par PREF et préfixe la chaîne résultante avec le résultat de PREF. Ensuite, il traite la règle spécifiée par SUF et suffixe la chaîne résultante avec le résultat de SUF.
Si le modificateur de FILE est spécifié, il inscrit le résultat de la variable dans le fichier indiqué.

Exemple
drop_table TEMPL "DROP TABLE $objectPhysicalname;;";
